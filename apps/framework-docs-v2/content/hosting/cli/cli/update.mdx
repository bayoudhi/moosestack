# 514 cli update

Use `514 cli update` to update installed CLI binaries.

## Usage

```bash
514 cli update [OPTIONS] [TARGET]
```

## Arguments

- `[TARGET]`: CLI target to update. Omit to update both.
  - `514`
  - `moose`

## Options

:::include /shared/hosting/cli/global-options.mdx

## Examples

```bash
# Update both CLIs
514 cli update

# Update only 514 CLI
514 cli update 514

# Update only moose CLI
514 cli update moose
```
