# 514 stream

Use `514 stream` to list stream resources in a deployment.

Plural alias also works: `514 streams`.

## Usage

```bash
514 stream [OPTIONS] <COMMAND>
```

## Subcommands

### List

```bash
514 stream list [OPTIONS] <DEPLOY_ID>
```

Arguments:
- `<DEPLOY_ID>`: Deployment ID.

Options:
:::include /shared/hosting/cli/resource-list-options.mdx

## Examples

```bash
514 stream list dep_abc123 --project my-org/my-project
514 streams list dep_abc123 --project my-org/my-project
```
