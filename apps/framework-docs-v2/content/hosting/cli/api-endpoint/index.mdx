# 514 api-endpoint

Use `514 api-endpoint` to list API endpoint resources in a deployment.

Plural alias also works: `514 api-endpoints`.

## Usage

```bash
514 api-endpoint [OPTIONS] <COMMAND>
```

## Subcommands

### List

```bash
514 api-endpoint list [OPTIONS] <DEPLOY_ID>
```

Arguments:
- `<DEPLOY_ID>`: Deployment ID.

Options:
:::include /shared/hosting/cli/resource-list-options.mdx

## Examples

```bash
514 api-endpoint list dep_abc123 --project my-org/my-project
514 api-endpoints list dep_abc123 --project my-org/my-project
```
