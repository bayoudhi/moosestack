# 514 workflow

Use `514 workflow` to list workflow resources in a deployment.

Plural alias also works: `514 workflows`.

## Usage

```bash
514 workflow [OPTIONS] <COMMAND>
```

## Subcommands

### List

```bash
514 workflow list [OPTIONS] <DEPLOY_ID>
```

Arguments:
- `<DEPLOY_ID>`: Deployment ID.

Options:
:::include /shared/hosting/cli/resource-list-options.mdx

## Examples

```bash
514 workflow list dep_abc123 --project my-org/my-project
514 workflows list dep_abc123 --project my-org/my-project
```
