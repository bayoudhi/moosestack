# 514 stream-to-table-sync

Use `514 stream-to-table-sync` to list stream-to-table sync resources in a deployment.

Plural alias also works: `514 stream-to-table-syncs`.

## Usage

```bash
514 stream-to-table-sync [OPTIONS] <COMMAND>
```

## Subcommands

### List

```bash
514 stream-to-table-sync list [OPTIONS] <DEPLOY_ID>
```

Arguments:
- `<DEPLOY_ID>`: Deployment ID.

Options:
:::include /shared/hosting/cli/resource-list-options.mdx

## Examples

```bash
514 stream-to-table-sync list dep_abc123 --project my-org/my-project
514 stream-to-table-syncs list dep_abc123 --project my-org/my-project
```
