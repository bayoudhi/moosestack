# 514 org

Use `514 org` to inspect available organizations and switch your active organization context.
The docs route remains `/hosting/cli/orgs` for URL compatibility.

## Usage

```bash
514 org [OPTIONS] <COMMAND>
```

## Subcommands

### List

List organizations available to the current authenticated user.

```bash
514 org list
```

### Switch

Switch the active organization context.

```bash
514 org switch [ORG_REF]
```

- `[ORG_REF]`: Organization ID, slug, or name. Omit to choose interactively.
