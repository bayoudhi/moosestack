# 514 auth login

Use `514 auth login` to start browser-based sign-in.

## Usage

```bash
514 auth login [OPTIONS]
```

## Options

:::include /shared/hosting/cli/global-options.mdx

## Expected outcome

- Browser sign-in flow opens.
- CLI receives a valid session.

## Recovery

- If browser launch fails, open: `https://www.fiveonefour.boreal.cloud/login`
- Retry command: `514 auth login`
