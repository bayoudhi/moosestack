# 514 org switch

Use `514 org switch` to change the active organization context.

## Usage

```bash
514 org switch [ORG_REF] [OPTIONS]
```

## Arguments

- `[ORG_REF]`: Organization ID, slug, or name. Omit to choose interactively.

## Options

:::include /shared/hosting/cli/global-options.mdx

## Examples

```bash
514 org switch acme
514 org switch
```
