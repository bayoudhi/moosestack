# 514 view

Use `514 view` to list view resources in a deployment.

Plural alias also works: `514 views`.

## Usage

```bash
514 view [OPTIONS] <COMMAND>
```

## Subcommands

### List

```bash
514 view list [OPTIONS] <DEPLOY_ID>
```

Arguments:
- `<DEPLOY_ID>`: Deployment ID.

Options:
:::include /shared/hosting/cli/resource-list-options.mdx

## Examples

```bash
514 view list dep_abc123 --project my-org/my-project
514 views list dep_abc123 --project my-org/my-project
```
