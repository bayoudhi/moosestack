# 514 materialized-view list

Use `514 materialized-view list` to list materialized-view resources in a deployment.

## Usage

```bash
514 materialized-view list [OPTIONS] <DEPLOY_ID>
```

## Arguments

- `<DEPLOY_ID>`: Deployment ID.

:::include /shared/hosting/cli/resource-list-options.mdx

## Examples

```bash
514 materialized-view list dep_abc123 --project my-org/my-project
514 materialized-views list dep_abc123 --project my-org/my-project
```
