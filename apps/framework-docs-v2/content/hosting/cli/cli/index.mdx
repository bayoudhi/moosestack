# 514 cli

Use `514 cli` for local CLI binary management operations.

## Usage

```bash
514 cli [OPTIONS] <COMMAND>
```

## Subcommands

### Update

Update installed CLIs (`514` and/or `moose`).

```bash
514 cli update [OPTIONS] [TARGET]
```

See the dedicated page: [`514 cli update`](/hosting/cli/cli/update).
