# 514 doc show

Use `514 doc show` to display a documentation page by slug.

## Usage

```bash
514 doc show [OPTIONS] <SLUG>
```

## Arguments

- `<SLUG>`: Documentation page slug (for example, `moosestack/olap` or `guides/chat-in-your-app#overview`).

## Options

:::include /shared/hosting/cli/doc-common-options.mdx

Additional command-specific options:
- `--web`

## Examples

```bash
514 doc show moosestack/olap
514 docs show guides/chat-in-your-app#overview --web
```
